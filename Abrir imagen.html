import cv2

# Cargar una imagen
img = cv2.imread('gato.jpg')

Glosario de Funciones

import cv2
La biblioteca OpenCV se utiliza para procesamiento de imágenes y visión por computadora. Para usarla en Python, necesitas instalar el paquete opencv-python y luego importarlo como cv2.
cv2.imread()
cv2.imread es una función de la biblioteca OpenCV en Python que se utiliza para cargar imágenes desde un archivo. Es una herramienta fundamental para trabajar con imágenes, ya que permite leerlas y convertirlas en matrices NumPy, lo que facilita su manipulación y análisis.
cv2.imshow()
cv2.imshow es un método de la biblioteca OpenCV en Python que se utiliza para mostrar imágenes en una ventana. Es una herramienta muy útil para trabajar con procesamiento de imágenes y visión por computadora, ya que permite visualizar los resultados de las operaciones realizadas sobre las imágenes
cv2.waitKey(0)
Detiene la ejecución del programa y espera a que el usuario presione una tecla.
cv2.destroyAllWindows()
Cierra todas las ventanas que hayan sido abiertas por OpenCV.

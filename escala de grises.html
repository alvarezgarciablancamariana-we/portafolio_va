Código del Ejemplo

import cv2

img = cv2.imread('equipo.jpg') #devuelve una matriz NumPy que representa la imagen en formato BGR (Azul, Verde, Rojo).
gris = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)#Convierte la imagen de color (BGR) a escala de grises.

cv2.imshow('Original', img) #Muestra una ventana con el nombre "Original" que contiene la imagen a color.
cv2.imshow('Escala de Grises', gris)#Muestra otra ventana con el título "Escala de Grises",
cv2.waitKey(0)#El parámetro 0 significa “esperar indefinidamente”.

#Si pusieras un número, por ejemplo 1000, esperaría 1000 milisegundos (1 segundo) antes de cerrar las ventanas automáticamente."""
cv2.destroyAllWindows()

Glosario de Funciones

import cv2
Importa la biblioteca OpenCV, utilizada para el procesamiento y análisis de imágenes y videos.
cv2.imread()
Carga una imagen desde el disco y la devuelve como una matriz NumPy que representa los valores de color de cada píxel.
cv2.cvtColor()
Convierte una imagen de un espacio de color a otro (por ejemplo, de color BGR a escala de grises).
cv2.imshow()
Muestra una imagen en una ventana con un título específico.
cv2.waitKey()
Detiene la ejecución del programa y espera a que el usuario presione una tecla antes de continuar.
cv2.destroyAllWindows()
Cierra todas las ventanas abiertas por OpenCV.

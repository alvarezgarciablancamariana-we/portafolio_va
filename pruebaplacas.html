Código del Ejemplo

import cv2
import pytesseract

# Ruta del ejecutable de Tesseract
pytesseract.pytesseract.tesseract_cmd = r"C:\Program Files\Tesseract-OCR\tesseract.exe"

# 1. Cargar imagen
img = cv2.imread(""placa1.jpg")

# 2. Obtener tamaño
alto, ancho, _ = img.shape

# 3. Zona del recorte (letras grandes)
y1 = int(alto * 0.25)
y2 = int(alto * 0.75)
x1 = int(ancho * 0.15)
x2 = int(ancho * 0.95)
recorte = img[y1:y2, x1:x2]

# --- PREPROCESAMIENTO MEJORADO ---

# Convertir a gris
gris = cv2.cvtColor(recorte, cv2. COLOR_BGR2GRAY)

# Aumentar contraste (truco muy efectivo)
gris = cv2.convertScaleAbs(gris, alpha=2.0, beta=0)

# Eliminar ruido
blur = cv2. GaussianBlur(gris, (5, 5), 0)

# Umbral Otsu (mejor para texto grueso)
_, binaria = cv2.threshold(blur, 0, 255, cv2. THRESH_BINARY_INV + CV2. THRESH_OTSU)

# Aumentar tamaño antes de OCR
binaria_grande = cv2.resize(binaria, None, fx=2, fy=2, interpolation=cv2. INTER_CUBIC)

# --- OCR con whitelist y modo "texto sencillo" ---
config = r'-c tessedit_char_whitelist=ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789 --psm 7'texto

cv2.waitKey(0)

Glosario de Funciones

cv2.imread()
Carga una imagen desde un archivo.
cv2.cvtColor()
Convierte una imagen de un espacio de color a otro. Muy útil porque OpenCV lee imágenes en formato BGR pero muchas bibliotecas usan RGB.
cv2.convertScaleAbs()
Aumenta o reduce el contraste y brillo de una imagen.
cv2.GaussianBlur()
Aplica un desenfoque Gaussiano para eliminar ruido.
cv2.threshold()
Convierte una imagen en escala de grises a una imagen binaria mediante un umbral.
cv2.resize()
Cambia el tamaño de una imagen ampliándola o reduciéndola.
pytesseract.image_to_string()
Realiza OCR y devuelve el texto detectado en la imagen.
cv2.imshow()
Muestra una imagen en una ventana.
cv2.waitKey()
Espera a que el usuario presione una tecla para continuar.
cv2.destroyAllWindows()
Cierra todas las ventanas mostradas con OpenCV.

import cv2 #importa pythoncv para manipular
import pytesseract

pytesseract.pytesseract.tesseract_cmd=r"C:\Users\MATUTINO\AppData\Local\Programs\Tesseract-OCR\tesseract.exe"

imgtexto=cv2.imread("texto1.webp")

#redimencionar el tamaño de la imagen
imgfinal=cv2.resize(imgtexto,(800,400))

#convertir a escala de grises
img_gris=cv2.cvtColor(imgfinal,cv2.COLOR_BGR2GRAY)

#separamos el fondo con el texto
img_res=cv2.threshold(img_gris,120,255,cv2.THRESH_BINARY)[1]
#devuelve los valores, los pixeles y la imagen procesada

#configurar OCR
texto=pytesseract.image_to_string(img_res,lang='spa')
#como parametro le pasamos la variable con la image44n proce4sada y el idioma
#imprimimos el titulo
print("texto detectado")
print(texto)

cv2.imshow("imagen analizada",imgfinal)
cv2.waitKey(0)
cv2.destroyAllWindows()

Glosario de Funciones

import pytesseract
Reconocimiento optico de caracteres.
cv2.resize()
Cambia el tamaño de la imagen.
cv2.cvtColor
Convierte la imagen a otro formato de color (por ejemplo, a escala de grises).
cv2.imread
Funcion de opencv que lee una imagen desde un archivo. Soporta multiples formatos como GPG, PNG, BMP, etc.
cv2.imshow
Funcion de opencv que lee una imagen en una ventana nueva. Requiere cv2.waitkey() para mantener la ventana abierta
cv2.threshold()
Separa el texto del fondo usando un umbral (convierte a blanco y negro).
pytesseract.image_to_string()
Reconoce el texto dentro de una imagen y lo convierte en texto digital.
pytesseract.pytesseract.tesseract_cmd
Indica la ruta donde está instalado el programa Tesseract.

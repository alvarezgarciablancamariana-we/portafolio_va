Código del Ejemplo

import cv2
from coordenadas2 import recortes
from movimientos import movimiento
from zoom_img import zoom
while True:
    opcion=int(input("PROCESAMIENTO DE IMAGENES \n1. Recortar\n2. Girar"
                     "\n3. Zoom\n4. Salir\nSelecciona una opcion: "))

    if opcion==1:
        print("Ejecuta el archivo coordenadas")
        recortes()
    if opcion==2:
        print("Ejecuta el archivo movimientos")
        movimiento()
    if opcion==3:
        print("Ejecuta el archivo zoom")
        zoom()
    elif opcion==4:
        print("Saliste del programa")
        break

Glosario de Funciones

from coordenadas2 import recortes
Importa la función recortes() desde un archivo externo llamado coordenadas2.py. Esta función, según su nombre, probablemente permite recortar partes de una imagen usando coordenadas.
from movimientos import movimiento
Importa la función movimiento() desde movimientos.py. Se usa para realizar rotaciones o giros en una imagen.
from zoom_img import zoom
Importa la función zoom() desde zoom_img.py. Permite aplicar un efecto de acercamiento o alejamiento sobre una imagen.
int()
Convierte un texto (string) en número entero. Es necesario porque input() devuelve texto, y se necesita comparar con números (if opcion == 1:).
if / elif
Evalúan condiciones y ejecutan un bloque de código según el valor de una variable. if → primera condición elif → condiciones adicionales
__name__ == "__main__"
Aunque no aparece directamente, los archivos importados (coordenadas2, movimientos, zoom_img) suelen usar esta línea para que su código principal solo se ejecute si el archivo se ejecuta directamente, no cuando se importa.

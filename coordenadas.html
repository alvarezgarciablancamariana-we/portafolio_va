Código del Ejemplo

import cv2
#importa la libreria opencv

#carga imagen
gato=cv2.imread('ga.jpg')

while True:

    recorte=gato[90:250,50:270] #alto y ancho
    cv2.imshow('imagen original',gato)
    cv2.imshow('imagen recortada',recorte)


    tecla=cv2.waitKey(0)#guarda en la variable tecla la tecla presionada
    if tecla==ord('y'):
        cv2.destroyAllWindows()
    elif tecla==ord('a'):
        cv2.imshow('imagen original',gato)
    elif tecla==ord('x'):
        cv2.imshow('imagen recortada',recorte)

        break

Glosario de Funciones

cv2.imread()
Carga una imagen desde el disco y la devuelve como una matriz NumPy con los valores de color de cada píxel.
while True:
Crea un bucle infinito que mantiene el programa en ejecución hasta que se cumpla una condición de salida.
gato[90:250, 50:270]
Realiza un recorte de la imagen seleccionando una región específica mediante coordenadas de alto y ancho.
ord()
Devuelve el código numérico (ASCII) correspondiente a un carácter dado.
break
Interrumpe el ciclo while y finaliza su ejecución
cv2.destroyAllWindows()
Cierra todas las ventanas abiertas por OpenCV.

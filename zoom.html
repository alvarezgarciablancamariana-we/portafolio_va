Código del Ejemplo

def zoom():
    import cv2

    img_original=cv2.imread("perro2.jpg")

    zoom=1.0

    while True:
        img=cv2.resize(img_original,None,fx=zoom,fy=zoom)
        cv2.imshow("Imagen nueva",img)
        tecla=cv2.waitKey(0)
        if tecla==ord("+"):
            zoom=zoom*1.2
        elif tecla==ord("-"):
            zoom=zoom*0.8
        elif tecla==ord("o"):
            zoom=1.0
        elif tecla==ord("x"):
            break

    cv2.destroyAllWindows()

if __name__=="__main__":
    zoom()

Glosario de Funciones

def zoom():
Define una función llamada zoom. Todo el código dentro de ella se ejecutará cuando se llame con zoom().
import cv2
Importa la librería OpenCV, que sirve para procesar imágenes y video en Python. Es necesaria para usar funciones como imread(), resize(), imshow(), etc.
cv2.imread("perro2.jpg")
Lee una imagen desde el archivo especificado (en este caso "perro2.jpg") y la carga en memoria como una matriz de píxeles.
cv2.resize(img_original, None, fx=zoom, fy=zoom)
Redimensiona (escala) la imagen original. Los parámetros fx y fy indican el factor de zoom (aumento o reducción).
cv2.imshow("Imagen nueva", img)
Muestra una ventana con el título "Imagen nueva" y enseña la imagen actual. Cada vez que cambias el zoom, la ventana se actualiza con la nueva versión.
cv2.waitKey(0)
Espera a que el usuario presione una tecla. El valor devuelto representa el código ASCII de la tecla presionada. Si se usa 0, espera indefinidamente hasta que se pulse una tecla.
ord("+")
Convierte un carácter (como "+", "-", "o", "x") en su valor numérico (ASCII). Sirve para comparar la tecla presionada con el valor que devuelve cv2.waitKey().
if __name__ == "__main__":
Bloque especial en Python que se ejecuta solo si el archivo se ejecuta directamente (no si se importa como módulo). Aquí se usa para ejecutar la función zoom() automáticamente cuando corres el script.
zoom = zoom * 1.2 / zoom = zoom * 0.8
Modifica el nivel de zoom: * 1.2 → aumenta un 20% el tamaño. * 0.8 → reduce un 20% el tamaño.
